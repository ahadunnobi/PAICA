# PAICA — Personal AI Communication Agent (Legacy)

A production-ready system to manage, classify, and reply to messages across multiple platforms in your personal style.

## 📁 Repository Structure
- **[legacy/](file:///c:/projects/PAICA/legacy/)**: Previous Next.js frontend and FastAPI backend (Docker-based).
- **[business.md](file:///c:/projects/PAICA/business.md)**: New business strategy for the mobile-first pivot.
- **[mobile/](file:///c:/projects/PAICA/mobile/)**: Real-time React Native application with on-device AI integration (In Development).

---

## 🚀 Legacy System (Getting Started)
The legacy system is preserved for archival purposes. To run it:
1. Navigate to `legacy/`.
2. Follow the instructions in `legacy/README.md` (if move didn't break paths) or use `docker-compose up --build` from the root of the legacy folder.

---

## 🧠 Future Vision: Mobile & On-Device AI
The new direction for PAICA focuses on:
1. **100% Privacy**: No message data ever leaves your phone.
2. **Offline Intelligence**: Using Google’s Gemma 2B via MediaPipe for on-device inference.
3. **Cross-Platform**: React Native supporting iOS and Android.
